# Image Instructions

To add your concrete product images to the website:

## Option 1: Use Local Images
1. Save your 8 images in this `images` folder with these names:
   - `concrete-slabs.jpg` (stacked precast slabs)
   - `block-wall.jpg` (concrete block wall)
   - `landscaping.jpg` (decorative landscaping with planters)
   - `wall-panel.jpg` (precast wall panel)
   - `countertop.jpg` (concrete kitchen countertop)
   - `decorative.jpg` (decorative finials/ornaments)
   - `bench.jpg` (concrete bench)
   - `steps.jpg` (concrete steps with planters)

2. Update the image paths in `index.html` from:
   ```html
   <img src="https://i.imgur.com/placeholder1.jpg" ...>
   ```
   to:
   ```html
   <img src="images/concrete-slabs.jpg" ...>
   ```

## Option 2: Use Image Hosting (Recommended for web deployment)
1. Upload your images to a service like:
   - Imgur (https://imgur.com)
   - Cloudinary (https://cloudinary.com)
   - Your hosting provider

2. Replace the placeholder URLs in `index.html` with your actual image URLs
